Message Limits: 160 Characters, 20 Words

Three word-limit strategies plus the character cap, all side by side. Type in a message and watch the outputs update.

Message input

0 chars 0 words fits 160

160 character cap

Returns the message if it is 160 characters or less; otherwise trims to the first 160 characters.

Solution 1: limit_words (optimal)

Split once and return both the allowed portion and remainder. 

def limit_words(message, limit=20):
    return ' '.join(message.split(maxsplit=limit)[0:limit]), ' '.join(message.split(maxsplit=limit)[limit:])

Solution 2: manual (no split)

Scan characters and build words manually.

def manual(message, limit=20):
    words=[]
    current=''

    for char in message:
        if char.isspace():
            if current:
                words.append(current)
                current = ''
                if len(words)==limit:
                    break
        else:
            current+=char
    if current and len(words) < limit:
        words.append(current)
    return ' '.join(words)

Solution 3: recursive

Recursively build the allowed words.

def limit_words_recursive(words, limit):
    if not words or limit == 0:
        return []
    return [words[0]] + limit_words_recursive(words[1:], limit-1)

def wrapper(text, limit=20):
    return " ".join(limit_words_recursive(text.split(), limit))
